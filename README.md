 COVID-19 Data Analysis & Dashboard System

 Project Overview

The COVID-19 pandemic generated a large amount of real-time data across the world. Analyzing this data manually is difficult and time-consuming. This project focuses on performing Exploratory Data Analysis (EDA) and developing an interactive dashboard to visualize COVID-19 trends using a real-world Kaggle dataset.

The dashboard helps users understand the spread of COVID-19, compare countries, analyze trends, and gain meaningful insights through interactive visualizations.



 Problem Statement

The global spread of COVID-19 created an unprecedented health crisis, generating massive volumes of data including confirmed cases, deaths, recoveries, and active cases. Organizations such as the World Health Organization (WHO) and research institutions continuously publish COVID-related datasets.

There is a need for an automated system that can:

* Analyze real-world COVID-19 data
* Identify trends and patterns
* Provide interactive visual insights
* Support data-driven decision making



 Objectives

1. Load and analyze COVID-19 dataset from Kaggle.
2. Perform data cleaning and preprocessing.
3. Generate descriptive statistics.
4. Analyze COVID-19 trends over time.
5. Compare affected countries based on cases, deaths, and recoveries.
6. Build an interactive dashboard using Plotly Dash.
7. Study relationships between different COVID-19 metrics.
8. Visualize data using various charts and graphs.
9. Generate insights and conclusions from the analysis.

---

Dataset Information

Dataset Source: Kaggle

Dataset Name: Corona Virus Report (COVID-19 Dataset)

Selected Columns:

* Date
* Country/Region
* Confirmed
* Deaths
* Recovered
* Active

Removed Columns:

* Province/State
* Latitude
* Longitude
* Last Update
* Incident Rate
* Case Fatality Ratio


Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Dash



 Data Preprocessing

The following preprocessing steps were performed:

* Handling missing values
* Removing duplicate records
* Converting date columns into proper datetime format
* Selecting relevant columns
* Validating numerical data



 Exploratory Data Analysis (EDA)

The project includes:

 Descriptive Statistics

* Total confirmed cases
* Total deaths
* Total recoveries
* Statistical summaries

Trend Analysis

* Daily confirmed cases
* Pandemic growth patterns
* Peak periods and waves

Country-Based Analysis

* Most affected countries
* Comparison of confirmed, recovered, and death cases

 Relationship Analysis

* Confirmed vs Deaths
* Confirmed vs Recovered
* Active vs Confirmed

 Correlation Analysis

* Heatmap visualization
* Correlation between COVID-19 metrics



 Dashboard Features

The interactive dashboard includes:

* Country Selection Dropdown
* Dynamic Line Chart
* Bar Chart
* Pie Chart
* Scatter Plot
* Real-time filtering by country



Visualizations

The dashboard provides:

1. Line Chart – COVID-19 cases over time
2. Bar Chart – Top affected countries
3. Pie Chart – Distribution of cases
4. Scatter Plot – Relationship analysis
5. Heatmap – Correlation analysis



 Expected Outcomes

* Better understanding of COVID-19 spread patterns
* Identification of high-risk countries
* Analysis of recovery trends
* Data-driven insights through interactive visualization



 Conclusion

This project demonstrates how data analytics and visualization can be used to understand pandemic trends effectively. By combining
Exploratory Data Analysis (EDA) with an interactive Plotly Dash dashboard, users can explore COVID-19 data, identify patterns, compare countries, and gain meaningful insights that support 
informed decision-making.
